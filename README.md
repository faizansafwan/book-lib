# Book Library System — Frontend

This is the **Angular frontend** for the Book Library System.  
It provides a modern, responsive UI to manage books, including listing, searching, adding, editing, and deleting entries.

---

## Tech Stack

- **Angular** (standalone components)
- **TypeScript**
- **Angular Signals**
- **HTML / CSS**
- **REST API integration**

## Install dependencies

```bash
npm install
```

## Run the development server

```bash
ng serve
```

The app will be available at:

```bash
http://localhost:4200
```
